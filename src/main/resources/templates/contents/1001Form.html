1.1 네트워크 및 공유 센터 이동
● 찾기 바를 통해서 제어판을 검색하여 제어판 열기
● 제어판에서 네트워크 및 공유 센터로 이동


1.2 이더넷 속성 확인
● 네트워크 및 공유 센터에서 이더넷 클릭
● 이더넷 상태 창에서 속성으로 이동


1.3 IPv4 속성 값 확인
● 이더넷 속성 창에서 인터넷 프로토콜 버전 4(TCP/IPv4) 선택


1.4 IP 주소 이상 여부 확인 및 수정
● 디지털혁신팀에서 발급해준 IP 주소가 맞는지 확인
 - 인터넷이 필요한 모든 프로그램 및 시스템에 이상이 존재할 시, 우선적으로 아래의 네트워크를 확인하도록 한다.
● 보통 기본 게이트웨이는 본인의 IP 주소에서 마지막 숫자에 1을 입력
● 당사의 서브넷 마스크의 구조는 255.255.255.xxx의 형태
● DNS 서버 주소는 아래와 같이 입력
 - 기본 DNS 서버: 192.168.65.77
 - 보조 DNS 서버: 168.126.63.1
